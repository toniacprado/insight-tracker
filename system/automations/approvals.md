# Automation Approvals
*Version:* v0.4  
*Date:* 2026-03-11  
*Last reviewed:* 2026-03-11

Use this file if the repo will automate actions.

Minimum default:
- destructive actions require explicit approval
- external sends require human review first
- sensitive summaries require audience checks before publication
- automated writes should leave a reviewable trace or artifact
- model-default or permission changes should be reviewed like any other guardrail change
