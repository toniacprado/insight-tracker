# Work Directory
*Version:* v0.4  
*Date:* 2026-03-11  
*Last reviewed:* 2026-03-11

Use this directory to keep task state visible across chats, sessions, and teammates.

Rules:
- `ACTIVE_TASKS.md` is the canonical current queue
- `items/` contains detailed task files when a task needs more than one line
- `LEARNINGS.md` captures durable discoveries that should survive the current session
- agents should update task state before ending meaningful work
- if a task is blocked, record the exact unblock needed
